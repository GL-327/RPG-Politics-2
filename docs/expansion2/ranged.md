# Expansion 2 — Ranged & Magic Weapons

Phase-2 **mixin-free** ranged/magic arsenal: bows, crossbows, chakrams, arc guns, wands,
staves, tomes, orbs, and domain focus items. All items use **Skyblock stats only** (no vanilla
attack/armour) and expose a **RIGHT CLICK** spell via Fabric `UseItemCallback`.

- **Package:** `com.political.expansion2.ranged`
- **Item ids:** prefixed `arc2_` (e.g. `arc2_solar_bow`)
- **Entry point:** `RangedExpansion2.register()` and `RangedExpansion2.items()`
- **Phase 1 isolation:** does not touch `com.political.expansion.ranged` or shared ability enums

## Integration checklist (for the integration agent)

1. **Register the cast engine** in `RpgPoliticsMod#onInitialize`:

   ```java
   com.political.expansion2.ranged.RangedExpansion2.register();
   ```

2. **Creative tab** — add stacks from `RangedExpansion2.items()` to `CreativeCatalog#gear` (or a dedicated tab):

   ```java
   for (net.minecraft.world.item.ItemStack s : com.political.expansion2.ranged.RangedExpansion2.items()) out.accept(s);
   ```

3. **Translations** — merge `tools/lang-fragments/ranged2.json` into `assets/politicalserver/lang/en_us.json`.

4. **Textures/models** — generated by `tools/gen-ranged2.js` into `textures/item/arc2_*.png`,
   `models/item/arc2_*.json`, `items/arc2_*.json`. Regenerate:

   ```bash
   cd tools && node gen-ranged2-java.js && node gen-ranged2.js
   ```

## How casting works

- Right-click with an `arc2_` weapon in the main hand fires its bound `RangedCast2`.
- Cooldowns and Mana are tracked in `RangedAbilityEngine2` (key prefix `|2|` avoids collision with Phase 1).
- **Intelligence scaling:** `power × (1 + max(0, (maxMana − 100) / 300))`.
- Targeted **BEAM** and **CHAIN** casts refund Mana when no target is found.
- Items carry `arc2_cast` in `custom_data`; tooltips are built by `RangedTooltip2`.

## Design notes

- **No mixins, no shared-enum edits.** Uses `arc2_cast` marker separate from Phase 1's `arc_cast`.
- **Hitscan + particles + lightning + summons** — same robust approach as Phase 1 and `PowerManager`.
- Spell variety inspired by RPG mage packs (Prominence II / JJK-style energy weapons).
